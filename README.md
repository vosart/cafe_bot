# 🍽️ Cafe Bot — Telegram бот для кафе

Telegram бот для автоматизации работы кафе с AI-ассистентом.

## Возможности

- 📋 Показ меню и часов работы
- 📅 Онлайн-бронирование столиков
- 🤖 AI-ответы на вопросы гостей (Yandex GPT)
- 👨‍💼 Панель администратора с просмотром броней и статистикой

## Технологии

- Python 3.11
- pyTelegramBotAPI
- SQLite
- Yandex Cloud API (YandexGPT)
